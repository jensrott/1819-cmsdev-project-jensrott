# 1819-cmsdev-project-jensrott

## Installatie Front-end
```
PS> git clone https://github.com/jensrott/1819-cmsdev-project-jensrott.git
PS> cd files->angular->sleep-app
PS> npm install
PS> ng serve
Go to localhost:4200
```

## Installatie Backend
```
PS> git clone https://github.com/jensrott/1819-cmsdev-project-jensrott.git
PS> cd files->wordpress->wordpress
PS> Create an empty sql database and use my backup_db.sql database. 
PS> Change wp-config with your own database data.
PS> Use a local webserver like MAMP to launch wordpress.
```
